VDFPy

VDFPy is a velocity distribution function classifier in Python.

Installation

python -m pip install vdfpy

Usage

We currently support extracting data from simulation (Vlasiator, FLEKS) and satellite observation (pyspedas). Please refer to the documentation for more details.
